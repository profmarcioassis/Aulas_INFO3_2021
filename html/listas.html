Listas em HTML

Listas ordenadas/numeradas: permite a criação de listas ordenadas/numeradas/classificadas

Utilitiza-se os elementos ol e li

O atributo type define o tipo da lista

O atributo start define o valor inicial da lista

Listas não ordenadas: permite a criação de listas com marcadores, ou seja, não numeradas

Utiliza-se os elementos ul e li

a propriedade list-style-type define o formato da lista e é inserida dentro do atributo style
