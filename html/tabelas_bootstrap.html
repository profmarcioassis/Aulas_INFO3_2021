Tabelas em HTML

  • A tag <TABLE> define uma tabela
  • A tag <TR> define uma linha para a tabela
  • A tag <TD> define uma célula para a linha
  • A tag <TH> define uma célula de títulos para a linha. Formata o texto negrito e centralizado, por padrão.
  • O atributo COLSPAN define mesclagem de colunas
  • o atributo ROWSPAN define mesclagem de linhas
  • A tag <caption> define uma legenda para a tabela
