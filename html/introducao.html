Introdução a HTML

HTML é uma linguagem de marcação de hypertexto, com a finalidade de estruturar as páginas web.

Parágrafo - Elemento P

O elemento p define um parágrafo e deve ser fechado /p.

Cabeçalho - Elemento h

O elemento h define cabeçalho (títulos) para partes do site. O h tem 6 níveis de cabeçalho.

Tanto o p quanto o h, tem quebra de linha.

Links - Elemento a

O elemento a permite ligar uma página a outra. Existem dois tipos de links

Interno: vincula para arquivos/páginas do próprio site

Externos: vincula com urls externas

Exemplos:

Página 2 IFMG Ouro Branco

Imgens - Elemento img

O elemento img permite a inserção de imagens na sua página web.

O atributo src define o arquivo a ser inserido.

O atributo alt define um texto alternativo para a imagem.

O atributo title define o rótulo quando o mouse passar sobre a imagem

As propriedades width e height definem a largura e a altura da imagem, neste caso.
