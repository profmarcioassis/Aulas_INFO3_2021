Introdução ao CSS

  • CSS é um padrão de formatação para páginas web
  • Proporciona para os desenvolvedores mais opções para formatação dos websites
  • Facilita na criação e manutenção dos layouts
  • O atributo style nos permite inserir propriedades CSS inbutida no HTML (inline)
  • O elemento style nos permite criar propriedades CSS de forma interna, dentro da seção head
  • Prioridades do CSS: inline > interno > externo

Sintaxe do CSS

seletor {
propriedade: valor da propriedade;
propriedade: valor da propriedade;
}

ID CSS

  • é um tipo de seletor bastante usado
  • é um seletor único para cada elemento.
  • não se pode atribuir um mesmo id para mais de um elemento html
  • sintaxe CSS: #nomedoid{propriedade: valor;}
  • no HTML, para referenciar o id, utiliza-se o atributo id="nome_do_id"

Parágrafo demonstrando a referencia ao id

Class CSS

  • é um tipo de seletor bastante usado
  • é um seletor para um grupo de elemento
  • uma classe pode ser referenciada por vários elementos HTML
  • sintaxe CSS: .nomedaclasse{propriedade: valor;}
  • para referenciar uma classe no HTML, basta usar o atributo class="nome das classes"
  • um elemento pode referenciar mais de uma classe, separando por espaço os nomes das classes

Formas de utilizar o CSS

  • In-line: utiliza-se o atributo style para embutir css no html
  • Interno: cria-se uma seção (atributo style) style dentro do head
  • Externo: cria-se um arquivo css separado (extensão .css)
    • é o mais recomendado, principalmente quando se tem um volume maior de páginas
    • facilita a manutenção e alteração dos estilos
    • exemplo: se quiser modificar o layout de muitas páginas
