Tags de formatação

  • Tag b - texto em negrito
  • Tag strong - texto importante
  • Tag i - texto itálico
  • Tag em - texto enfatizado
  • Tag sub - Texto subescrito
  • Tag sup - texto em seperescrito

Exemplos

Texto negrito usando a tag B
Texto importante utilizando a tag STRONG
Texto itálico utilizando a tag I
texto enfatizado utilizando a tag EM
Texto subescrito utilizando a tag SUB. Ex.: H2O
Texto superescrito utilizando a tag SUP. Ex.: X3
