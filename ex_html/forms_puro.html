Formulários HTML

  • A tag <form>...</form> define a estrutura principal do formulário
  • O atributo action define qual a ação ao submeter o formulário
  • O atributo method define o método de envio do formulário
  • O atributo required define que um campo é obrigatório
  • O atributo autofocus seta o foco para o campo ao carregar a página
  • O atributo placeholder define um texto de orientação na input
